MathsBro is a simple command-line math quiz game written in Python.  
It helps users practice basic arithmetic operations with different difficulty levels and generates session reports in both text and HTML formats.

Features

- Multiple difficulty modes:
  - Demo (addition only)
  - Easy (addition & subtraction)
  - Medium
  - Hard (includes multiplication)
- Interactive CLI gameplay
- Score calculation with percentage
- Session reports saved as:
  - `.txt` file
  - `.html` file
- Unicode symbols for correct (✓) and wrong (✗) answers
  
Requirements

- Python 3.8 or higher
No external libraries required.
